# cat-mouse
Bạn chơi vai chuột (di chuyển trái/phải bằng phím mũi tên). Mèo sẽ đuổi theo bạn tự động (di chuyển về phía chuột). Trên đường có gai (đỏ) – nếu chạm, mất 1 tim. Có cục tim (hồng) để ăn – chạm để hồi 1 tim (tối đa 3 tim). Mục tiêu: Sống sót lâu nhất, tích điểm bằng cách ăn tim và tránh gai. Game over khi hết tim.
